# Maze / Cave / Q-Learning Visualizer

**C++20, Qt Widgets, Crow, HTML/CSS/JS, CMake, Makefile, Google Test, Doxygen**

Комплексный проект для генерации, визуализации и решения лабиринтов, моделирования пещер на клеточном автомате, обучения агента прохождению лабиринта методом **Q-learning**, а также предоставления базового **web-интерфейса** через сервер на **Crow**.

## Что реализовано

- Разработала приложение на **C++20** с графическим интерфейсом на **Qt Widgets**
- Реализовала генерацию **идеального лабиринта** по **алгоритму Эллера**
- Добавила загрузку и сохранение лабиринтов в файл в собственном формате
- Реализовала поиск кратчайшего пути в лабиринте с помощью **BFS**
- Добавила визуализацию найденного маршрута с анимацией отображения
- Реализовала отдельный модуль генерации **пещер** на основе **клеточного автомата**
- Добавила пошаговый и автоматический режим эволюции пещеры через `QTimer`
- Реализовала бонусный модуль **reinforcement learning**:
  - обучение агента на одном фиксированном лабиринте
  - выбор целевой точки
  - построение маршрута обученным агентом из произвольной стартовой клетки
  - собственная реализация **Q-learning** без готовых RL-библиотек
- Спроектировала интерфейс с отдельными вкладками:
  - **Maze**
  - **Cave**
  - **Q-Learning**
- Реализовала **web-интерфейс** для базовой работы с лабиринтом
- Подняла **HTTP-сервер на Crow** на C++
- Подготовила **HTML/CSS/JavaScript**-страницу для взаимодействия с серверной частью
- Реализовала обмен данными между C++ backend и web-страницей
- Подготовила **unit-тесты** для модулей:
  - генерации лабиринта
  - решения лабиринта
  - генерации пещеры
  - обучения агента
- Настроила сборку проекта через **CMake** и пользовательский запуск через **Makefile**
- Подготовила документацию к проекту с помощью **Doxygen**
